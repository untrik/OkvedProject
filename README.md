# Система управления компаниями и ОКВЭД

## Описание

Данное приложение представляет собой консольную утилиту для работы с компаниями и ОКВЭДами.

----------------------------------------------------------------------------------------------------------

## Предметная область
В рамках приложения рассматриваются две основные предметные области:
### Компании
Система хранит и обрабатывает информацию о компаниях:
идентификатор  
название  
ИНН  
капитал компании  
контактные данные (сайт, телефон, email)  
статус (активна / ликвидирована)  
основной и дополнительные ОКВЭДы

### Каталог ОКВЭД
секции (A, B, C и др.)  
группы (например, 01, 02)  
конкретные виды деятельности (например, 01.11)  

-----------------------------------------------------

## Функциональность

### Работа с каталогом ОКВЭД:
просмотр всех секций  
получение групп по секции  
получение ОКВЭДов по группе  
поиск ОКВЭДов по названию  

### Работа с компаниями:
добавление компаний  
назначение основного ОКВЭДа  
добавление дополнительных ОКВЭДов  
топ-5 компаний по капиталу  
доля активных компаний  
поиск компаний по секции  
поиск компаний по группе  
просмотр всех компаний
